## Installation Instructions

To run locally:

```bash
git clone https://github.com/dewe2513/dewe2513.github.io.git
cd dewe2513.github.io
```

Then open `index.html` in your browser, or run a local server:

```bash
python -m http.server 8000
```

Visit `http://localhost:8000` in your browser.

**Deploying changes:**

```bash
git add .
git commit -m "Update content"
git push origin main
```

Changes automatically deploy to GitHub Pages within 1-2 minutes.

## Use as Template

Want to create your own portfolio website? Fork this repository and customize it:

1. Click the "Fork" button at the top of this repository
2. Rename the forked repository to `yourusername.github.io`
3. Update the HTML files with your own information
4. Replace images in the `Images/` folder with your own
5. Enable GitHub Pages in your repository settings (Settings → Pages → Source: main branch)

Your site will be live at `https://yourusername.github.io`!
